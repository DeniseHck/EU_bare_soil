# EU_bare_soil
Developing a model to detect bare soil exposure across European arable land using Sentinel-2, Sentinel-1 and the LUCAS 2018-2022 picture database


## Methods
- Extract Sentinel-2 and Sentinel-1 time series across >4,000  agricultural points across Europe surveyed as part of LUCAS 2018-2022
- Train Machine Learning model to classify individual days into “bare” or “not bare”
